# aseprite file
The aseprite file has all the layers in folders, they should be all named accordingly. The paws are meant to go on top of the buttons ("how the buttons would work" layer for reference). Everything in the "cat" folder can stay together.
Toggling the visibility for the "Background" layer turns the background transparent.

# frame folders
I've added all the frames to the frames folder if they're needed. The paws are the same on every frame, so they're there only once and can be duplicated for every frame.

    All frames (transparent)
